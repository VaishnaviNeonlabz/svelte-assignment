# Svelte Assignment

A modern, responsive static 3-page company website built using **SvelteKit**, **TypeScript**, **Tailwind CSS**, and **Yarn**.

## 📌 Project Overview

This project was developed as part of a Svelte assignment. It includes three responsive pages with reusable Svelte components and a modern UI.

## ✨ Features

- Responsive Navigation Bar
- Home Page
- About Page
- Contact Page
- Reusable Components
- Tailwind CSS Styling
- Svelte Reactivity Example
- Smooth Scrolling
- Mobile Friendly Design

## 🛠️ Technologies Used

- SvelteKit
- TypeScript
- Tailwind CSS
- Yarn

## 📂 Project Structure

```text
src/
 ├── lib/
 │   └── components/
 ├── routes/
 │   ├── +page.svelte
 │   ├── about/
 │   └── contact/
 └── app.css
```

## 🚀 Installation

```bash
yarn install
yarn dev
```

## 🏗️ Build

```bash
yarn build
```
